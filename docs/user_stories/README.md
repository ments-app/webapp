## Structure

Each file covers one domain/epic. Stories follow the format:

```
US-[domain]-[number]: [Title]
As a [persona], I want to [action], so that [outcome].

Acceptance Criteria:
- [ ] ...

Edge Cases:
- ...
```

## Personas

| Persona | Description |
|---|---|
| **Founder** | Building or has built a startup. Wants visibility, co-founders, funding |
| **Builder** | Developer, designer, or operator. Looking for projects, gigs, or collaborations |
| **Explorer** | Browsing for opportunities, ideas, and people. No active project yet |
| **Investor** | Scouting startups and founders for potential investment |
| **Visitor** | Not logged in. Landed via link or search |
